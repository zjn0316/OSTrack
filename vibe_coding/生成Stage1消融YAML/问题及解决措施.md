# 问题及解决措施

| 问题 | 解决措施 |
|---|---|
| `best` 编码器和 `Tbest` 尚未通过实验确定 | 先按当前主候选 `tcn` 和 `T=10` 生成可运行 YAML，后续根据 Stage-1 分析结果替换对应字段 |
| Stage-1 训练入口缺少 `settings.use_wandb`，导致 `LTRTrainer` 初始化失败 | 在 `tracking/train_uwb.py` 中显式设置 `settings.use_wandb = False` |
| Stage-1 训练入口缺少 `settings.log_file`，导致第 1 个 epoch 打日志时报错 | 在 `tracking/train_uwb.py` 中创建 `output/logs` 并设置 `ugtrack-<config>.log` |
| `fail_safe=True` 会把训练异常包装成“Finished training”，后续才暴露 checkpoint 缺失 | 将 `tracking/train_uwb.py` 的 `trainer.train(..., fail_safe=False)`，让真实错误直接中断 |
