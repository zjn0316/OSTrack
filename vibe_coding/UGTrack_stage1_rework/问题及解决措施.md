# 问题及解决措施

- 问题：回滚后 UWB encoder 只剩 MLP/Conv1D，默认输出 768 维。
  解决：重写为 MLP/GRU/TCN，默认 TCN 输出 128 维，并保留 `conv1d` 到 TCN 的兼容别名。

- 问题：回滚后 Stage-1 仍使用 MSE + MSE。
  解决：训练入口根据配置选择坐标损失，默认 `l1`；置信度默认使用 `BCEWithLogitsLoss`。

- 问题：回滚后置信度 head 直接输出激活后的 `uwb_conf`。
  解决：改为输出 `uwb_conf_logit`，再通过 sigmoid 得到 `uwb_conf_pred`。
