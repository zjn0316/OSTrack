# 如何为 OSTrack 添加新数据集接口

本文档说明如何在 OSTrack 中添加一个新的数据集接口。

---

## 📋 总体流程

1. **创建数据集类文件** - 在 `lib/train/dataset/` 下创建新的 `.py` 文件
2. **注册数据集** - 在 `__init__.py` 中导入新数据集
3. **配置路径** - 在 `lib/train/admin/local.py` 中添加数据集路径
4. **注册数据集名称** - 在 `lib/train/base_functions.py` 的 `names2datasets()` 函数中添加映射
5. **创建配置文件** - 在 `experiments/ostrack/` 下创建 YAML 配置文件

---

## 1️⃣ 创建数据集类文件

### 文件位置
```
lib/train/dataset/your_dataset_name.py
```

### 必须实现的内容

#### A. 继承 BaseVideoDataset

```python
from .base_video_dataset import BaseVideoDataset
from lib.train.data import jpeg4py_loader
from lib.train.admin import env_settings

class YourDataset(BaseVideoDataset):
    """数据集描述"""
    
    def __init__(self, root=None, image_loader=jpeg4py_loader, split='train', data_fraction=None):
        """
        参数:
            root: 数据集根目录路径
            image_loader: 图像加载函数（默认 jpeg4py_loader）
            split: 'train', 'val', 或 'test'
            data_fraction: 使用数据的比例（None表示全部）
        """
        # 从环境配置获取路径
        root = env_settings().your_dataset_dir if root is None else root
        super().__init__('YourDatasetName', root, image_loader)
        
        self.split = split
        # 初始化数据集
        self.sequence_list = self._build_sequence_list()
        self.seq_per_class = self._build_class_list()
```

#### B. 必须实现的方法

| 方法 | 说明 | 返回值 |
|------|------|--------|
| `get_name()` | 返回数据集名称 | `str` |
| `get_num_sequences()` | 返回序列数量 | `int` |
| `get_sequence_info(seq_id)` | 返回指定序列的标注信息 | `dict{'bbox', 'valid', 'visible'}` |
| `get_frames(seq_id, frame_ids, anno=None)` | 返回指定帧的图像和标注 | `(frame_list, anno_frames, object_meta)` |

#### C. 可选实现的方法

| 方法 | 说明 | 默认值 |
|------|------|--------|
| `has_class_info()` | 是否有类别信息 | `False` |
| `has_occlusion_info()` | 是否有遮挡信息 | `False` |
| `get_num_classes()` | 返回类别数量 | - |

#### D. 辅助方法（建议实现）

```python

    """获取单帧图像"""
```

## 4️⃣ 注册数据集名称映射

### 修改文件：`lib/train/base_functions.py`

#### A. 添加导入

在文件顶部添加数据集类的导入：

```python
from lib.train.dataset import Lasot, Got10k, ..., YourDataset
```

#### B. 在 names2datasets() 函数中添加映射

```python
def names2datasets(name_list: list, settings, image_loader):
    assert isinstance(name_list, list)
    datasets = []
    for name in name_list:
        # 添加到允许的数据集名称列表
        assert name in ["LASOT", "GOT10K_vottrain", ..., "YOUR_DATASET"]
        
        # ... 其他数据集的处理 ...
        
        # 添加你的数据集处理逻辑
        if name == "YOUR_DATASET":
            datasets.append(YourDataset(settings.env.your_dataset_dir, 
                                       split='train',  # 或根据名称动态设置
                                       image_loader=image_loader))
    
    return datasets
```

**示例**:
```python
if name == "OTB100_UWB":
    datasets.append(OTB100UWB(settings.env.otb100_uwb_dir, 
                             split='train',  # 会在配置文件中指定
                             image_loader=image_loader))
```

## 5️⃣ 创建训练配置文件

### 文件位置
```
experiments/ostrack/your_config.yaml
```

### 关键配置项

```yaml
DATA:
  TRAIN:
    DATASETS_NAME:
    - YOUR_DATASET  # 必须与 names2datasets 中的名称一致
    DATASETS_RATIO:
    - 1
    SAMPLE_PER_EPOCH: 10000  # 每个epoch的样本数
  VAL:
    DATASETS_NAME:
    - YOUR_DATASET
    DATASETS_RATIO:
    - 1
    SAMPLE_PER_EPOCH: 2000

TRAIN:
  BATCH_SIZE: 16  # 根据GPU显存调整
  EPOCH: 50
  NUM_WORKER: 4
  # ... 其他训练参数 ...
```

## 🔍 常见问题

### Q1: 如何处理不同的标注格式？

在 `_read_bb_anno()` 方法中根据实际格式解析：

```python
# CSV 格式（逗号分隔）
gt = pandas.read_csv(bb_anno_file, delimiter=',', header=None).values

# TXT 格式（空格或Tab分隔）
gt = pandas.read_csv(bb_anno_file, delimiter='\t', header=None).values

# 自定义格式
with open(bb_anno_file, 'r') as f:
    lines = f.readlines()
    gt = [list(map(float, line.strip().split(','))) for line in lines]
```

### Q2: 如何处理图像命名格式？

在 `_get_image_path()` 方法中调整：

```python
# 8位数字: 00000001.jpg
img_filename = f'{frame_id:08d}.jpg'

# 5位数字: 00001.jpg
img_filename = f'{frame_id:05d}.jpg'

# 无前缀: 1.jpg
img_filename = f'{frame_id}.jpg'
```

### Q3: 如何使用多个数据集混合训练？

在配置文件中指定多个数据集：

```yaml
DATA:
  TRAIN:
    DATASETS_NAME:
    - LASOT
    - GOT10K_vottrain
    - YOUR_DATASET
    DATASETS_RATIO:
    - 1
    - 1
    - 1
```

### Q4: 调试数据集加载问题？

添加打印语句检查：

```python
def __init__(self, ...):
    print(f"Loading dataset from: {root}")
    print(f"Split: {split}")
    print(f"Number of sequences: {len(self.sequence_list)}")
```
