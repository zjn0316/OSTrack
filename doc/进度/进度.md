## 进度计算规则

总步骤数 = `## 完整步骤` 区域内的任务数量。

已完成步骤数 = `## 完整步骤` 区域内标记为 `[x]` 或 `[X]` 的任务数量。

进度百分比 = `已完成步骤数 / 总步骤数 * 100%`。

每一个步骤都会成为进度条上的一个节点：

- `●` 表示已完成。
- `○` 表示未完成。

## 使用方式

复制本模板内容，保存为 `进度.md`，然后运行：

```powershell
python -X utf8 .\progress.py
```

如果要解析其他文件，可以指定文件路径：

```powershell
python -X utf8 .\progress.py .\进度模板.md
```
