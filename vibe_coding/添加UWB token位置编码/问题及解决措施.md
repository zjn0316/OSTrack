# 问题及解决措施

## 2026-05-01

| 问题 | 解决措施 |
| --- | --- |
| 之前仅有 `uwb_token` 投影和拼接，没有 UWB token 专属位置编码 | 在 UGTrack 专用 backbone 中新增 `uwb_pos_embed`，并在拼接前执行 `uwb_token + uwb_pos_embed` |
| 不能污染官方 OSTrack | 只修改 `lib/models/ugtrack/vit_uwb.py` 和 `lib/models/ugtrack/vit_ce_uwb.py`，不改 `lib/models/ostrack/` |
| 位置编码不应参与 weight decay | 在两个 UGTrack backbone 的 `no_weight_decay()` 中加入 `uwb_pos_embed` |
